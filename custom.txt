# Add your custom entries here, one per line.
#
# Supported formats (same as v2fly domain-list-community):
#
#   domain:example.com       — exact domain match
#   full:sub.example.com     — full domain match

# Blank lines are also ignored.
#

#   domain:mystreaming.tv
#   full:watch.example.com
